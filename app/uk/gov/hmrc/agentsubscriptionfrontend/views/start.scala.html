Agent Services account: sign in or set up

The Agent Services account is for professional tax agents.

It allows agencies to:

  • connect to HMRC using commercial accounting software
  • access new HMRC online services

Setting up an account creates a new agent Government Gateway ID.

You must save this new User ID for your agency's records, even if your agency already has one or more.


To set up an account you need:

  • an existing agent Government Gateway ID - you will have at least one if you have used any HMRC agent services online before
  • your agency's Unique Taxpayer Reference (UTR) and associated postcode, if you are a sole trader or a member of a partnership
  • your agency's Corporation Tax reference number and associated postcode, if it is an incorporated company

If you have already set up an Agent Services account, use your new agent Government Gateway ID to sign in.

Before you start

If you're charging for services as an agent, you must be supervised for anti-money laundering.
